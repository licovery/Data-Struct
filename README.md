一些学习心得感想：网上大多数一搜就出来的资料，质量都比较一般，blog里面大多数的知识都不成体系，良莠不齐。如果只是想知道怎么用，可能看个blog差不多。但是要真正地深入地理解，最好的办法是官方文档和资料或者相关的书籍，例如C和C++的Reference和Linux的Man Page，书籍的话就C++ primer Unix编程系列。从看Blog都看官方文档，是一个由浅入深的过程，二者互补。
